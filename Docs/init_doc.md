# PlanPie Project Plan

## Overview
PlanPie is a web application that consists of various modules and features designed to help users stay productive.

## Core Features
1. **Notes App** - Digital note-taking and organization
2. **Pomodoro Clock** - Time management and focus sessions
3. **Calendar & Scheduler** - Event planning and task scheduling
4. **AI Task Agent** - Prioritizes tasks, schedules reminders, and adapts notifications based on urgency and user behavior
5. **Spotify Integration** *(optional)* - Music widget using Spotify API for users with Spotify accounts
6. **Blog Section** *(optional)* - Personal blogging functionality

## Application Flow
**Frontend → User Login → Dashboard → Feature Access**

1. **Landing Page** - Initial user entry point
2. **Authentication** - User login page with Google OAuth support
3. **Main Dashboard** 
   - Website name and logo (left side)
   - Settings option (right side) - opens settings page
   - Collapsible side panel for feature navigation
   - Task overview with due dates and times
   - Daily positive affirmation
4. **Feature Pages** - Individual pages for each module accessed via side panel

## Design Concept
### Light Mode
- **Color Palette**: Soft blues and purples (beach-inspired theme)
- **Typography**: Easy-to-read fonts
- **UI Elements**: Rounded edges throughout

### Dark Mode
- **Color Palette**: Neon blues with black or charcoal grey backgrounds
- **Consistent**: Maintains rounded edge design language
